Active cell

Active cell is a navigation feature.
If a cell is active, pressing ENTER will show the editor for the cell.
If a cell is active, holding SHIFT and using LEFT, RIGHT, UP, DOWN
will change the selection of cells.
Property to disable/enable active cell:
activeCell

Cells selection

It is possible to select cells and COPY or PASTE data.
Use CTRL + C and CTRL + V to copy or paste data.
