Your First Grid

We assume you included sources to page. 

<link href="https://cdn.jsdelivr.net/npm/fg-grid/styles/fg-grid.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/fg-grid/dist/fg-grid.min.js"></script>

Add div to page

Add a div in your webpage. Give it an id. 

<div id="container"></div>

The first one is id name.

Another one approach to provide dom.
