Sizing

There are two ways to set the size of the grid.

Container Size

The first option is to omit the width and height properties in the grid configuration.
In this case, the grid becomes responsive: it will automatically fill the size of its parent container
and update itself when the container resizes.
It's important to correctly define the size of the container.
This can be done using fixed or percentage-based values.

Fixed Container Size

<div id="container" style="width: 500px; height: 400px;"></div>
Responsive Container Size

<div id="container" style="width: 100%; height: 100%;"></div>

Grid Configuration

The second option is to explicitly set the size in the grid configuration.
The width and height must be provided as numbers (not strings). 


new Grid({
  width: 500,
  height: 400,
  ...
});

Combination

You can also combine both approaches: define either width or height in the grid config,
and the grid will get the missing size from the container.
