Understanding FG-Grid

Main properties

Main grid properties: columns and data.



Main config properties

data

Array

Grid data

Data in grid can be set in many ways, depending on needs.

 JSON data 

data: [
  { brand: "Lexus", model: "RX 350", price: 60000, year: 2021 },
  { brand: "Lexus", model: "NX 300", price: 50000, year: 2023 },
  { brand: "Toyota", model: "Land Cruiser Prado", price: 70000, year: 2022 },
  { brand: "Toyota", model: "RAV4", price: 35000, year: 2023 },
  { brand: "Volkswagen", model: "Tiguan", price: 38000, year: 2021 }
]

columns

Array

Grid columns.

 

columns: [{
  index: 'brand',
  title: 'Brand',
  type: 'string'
},{
  index: 'model',
  title: 'Model',
  type: 'string'
},{
  index: 'price',
  title: 'Price',
  type: 'number'
},{
  index: 'year',
  title: 'Year',
  type: 'number'
}]

defaults

Object

Columns default properties.

 

defaults: {
  type: 'string',
  width: 100,
  sortable: true
},

height

Number

Grid height.

 

height: 500,

width

Number

Grid width.

 

width: 400,
