Filter operators

There are several operators that are possible to use in the filter field.
They are:
Operator Description
< Less
> More
<= Less or equals
>= More or equals
= Contains
== Equals
!= Does not contain
!== Does not equal
empty null, undefined, empty string
!empty Is not one of null, undefined, empty string
a_ Starts with
_a Ends with
regex Follow regex rule
+ Positive
- Negative

The default operator is '='.

Example: add filter to grid


const columnPrice = grid.getColumn('price');
grid.filter(columnPrice, 35000, '>');

Example: default operator


const column = grid.getColumn('brand');
grid.filter(column, 'Toyota');
