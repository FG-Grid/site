rowGroupExpanded

Boolean

Provides list of expanded row.

Example: Array list


new Grid({
  rowGroupExpanded: ['Japan', 'Japan/Toyota', 'Japan/Toyota/Camry', 'UK', 'Sweden'],
  ...

Example: Function


new Grid({
  rowGroupExpanded: (groupName) => {)=>{
    switch(groupName){
      case 'Japan':
      case 'Japan/Toyota':
      case 'Japan/Toyota/Camry':
      case 'UK':
      case 'Sweden':
        return true;
    }

    return false;
  },
  ...

Example: Boolean

Expand all groups. 


new Grid({
  rowGroupExpanded: true,
  ...

Default


undefined
