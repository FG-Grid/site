Theming

Grid provides 3 themes out of the box: default, air, dark.
Also, you can design a custom theme.

Creating a custom theme is straightforward:
1 - Come up with a name for your theme (e.g., your company name, like Nike).
2 - Use the default theme's CSS as a starting point.
3 - Override CSS variables and write additional custom CSS if needed.
4 - Include your CSS theme file on the page.
5 - Set the theme property in the grid configuration to your theme name.

Let's study designing theme on how it was done dark theme.
