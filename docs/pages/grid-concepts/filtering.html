Filtering

Enable Filtering

To display a filter field in the column header, set filter: true and enable the filter bar by setting filterBar: true. 


new Grid({
  ...
  filterBar: true,
  columns: [{
    index: 'brand',
    title: 'Brand',
    filter: true
  },{
    ...
  }]
});

Filter Operators

Several operators can be used in the filter field:
<, >, <=, >=, =, !=

Methods

Sometimes you may need to apply filters programmatically instead of setting them in the grid config.
For example, when you have a form whose changes should trigger filtering.
In such cases, the following two methods can be used:

filter

(column: object, value: mixed, operator?: string)

The default operator is '='.

Example: filter by column and value


const grid = new Grid({
  ...,
  filterBar: true,
  columns: [{ index: 'name', filter: true }]
});

const column = grid.getColumn('name');
grid.filter(column, 'Nick');

Example: filter by value and operator


const grid = new Grid({
  ...
});

const columnPrice = grid.getColumn('price');
grid.filter(columnPrice, 30000, '>');

clearFilter

(column?: object, operator?: string)

Example: clear all filters


const grid = new Grid({
  ...
});

grid.clearFilter();

Example: clear filters by column index


const grid = new Grid({
  ...
});

grid.clearFilter('name');

Example: clear filters by column index and operator


const grid = new Grid({
  ...
});

grid.clearFilter('price', '<');
