filter(column: object, value: mixed, operator: '<'|'>'|'<='|'>='|'='|'!='): void

Add filter to grid

Filter operators

There are several operators that possible to use in filter field.
They are: <,>,<=,>=,=,!=

The default operator is '='.

Example: add filter to grid


const columnPrice = grid.getColumn('price');
grid.filter(columnPrice, 35000, '>');

Example: default operator


const column = grid.getColumn('brand');
grid.filter(column, 'Toyota');
