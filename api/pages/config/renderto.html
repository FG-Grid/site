renderTo

Mixed

Dom/id, where to render grid.

Example: String


<div id="container"></div>
...

new Grid({
  renderTo: 'container',
  width: 690,
  height: 350,
  ...

Example: Dom


<div id="container"></div>
...

new Grid({
  renderTo: document.querySelector('#container'),
  width: 700,
  height: 350,
  ...

Example: Argument

It is possible do not define renderTo property and provide dom element as first argument. 


<div id="container"></div>
...

new Grid(document.querySelector('#container'), {
  width: 700,
  height: 350,
  ...

Default


undefined
