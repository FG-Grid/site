flex

Number

Equivalent to flex-grow in CSS.
When flex is set on one or more columns, any width value is ignored and instead
the remaining free space in the grid is divided among flex columns in proportion to their flex value,
so a column with flex: 2 will be twice the size as one with flex: 1.

Example


columns: [{
  index: 'brand',
  flex: 2
},{
  index: 'model',
  flex: 1
},{
  index: 'year',
  width: 100
},{
  index: 'price',
  width: 100
}]

Default


undefined
