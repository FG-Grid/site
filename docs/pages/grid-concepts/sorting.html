Sorting

Local sorting

To enable local sorting, you need to set for columns that needs to be sorting, param sortable


new Grid({
  columns: [{
    index: 'brand',
    sortable: true
  },{
    index: 'model',
    sortable: true
  },{
    index: 'price',
    type: 'number',
    sortable: true
  },{
  ...
To avoid repeating this property for all columns, there is a default option property. 

new Grid({
  defaults: {
    sortable: true
  },
  columns: [{
    index: 'brand'
  },{
    index: 'model'
  },{
    index: 'price',
    type: 'number'
  },{
  ...
