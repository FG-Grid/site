Installation

In this document you will learn several approaches of including FG-Grid.
FG-Grid is Vanilla JavaScript library, it has no dependencies from other libraries.
FG-Grid has integration with React and TypeScript.

Include Grid

Include files from CDN

Include the JavaScript files in the <head> section of your web page as shown below.
FG-Grid uses public CDN - jsDelivr 

<link href="https://cdn.jsdelivr.net/npm/fg-grid/styles/fg-grid.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/fg-grid/dist/fg-grid.min.js"></script>

Alternatively load files from own domain

Download library from GitHub Latest Build and include files to your page. 

<link href="/fg-grid/styles/fg-grid.min.css" rel="stylesheet">
<script src="/fg-grid/dist/fg-grid.min.js"></script>

npm

Run the following command to start your download. 


npm install fg-grid

To use React wrapper install extra package 


npm install fg-grid-react-wrapper

Get Started

Now you are ready to use FG-Grid.
