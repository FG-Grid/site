flashChangesColors

[string, string]

By default, changes will be flashed. Grid parameter flashChanges is true by default. For flashing, parameter flashChangesColors is used. First color for usual cell, second color to selected rows.

Example


new Grid({
  flashChangesColors: ['#e0e5e9', 'beige']
  ...

Default


['#e0e5e9', 'beige']
