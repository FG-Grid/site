Sorting

Local sorting

To enable local sorting, you need to set for columns that needs to be sorting, param sortable


new Grid({
  columns: [{
    index: 'brand',
    sortable: true
  },{
    index: 'model',
    sortable: true
  },{
    index: 'price',
    type: 'number',
    sortable: true
  },{
  ...
To avoid repeating this property for all columns, there is a default option property. 

new Grid({
  defaults: {
    sortable: true
  },
  columns: [{
    index: 'brand'
  },{
    index: 'model'
  },{
    index: 'price',
    type: 'number'
  },{
  ...

clearSort

(column?: object, multi?: boolean)

The clearSort method clears sorting.
To clear sorting on a specific column, you need to pass the column.
Columns can be retrieved using getColumn or getColumnById.
Consider the multi argument when using multi-column sorting.
If multiple columns are sorted and you want to clear only one of them, use the multi argument.

Example: clear all sorters


grid.clearSort();

Example: clear exact column(getColumn)

Method getColumn returns column by data index. 


const columnPrice = grid.getColumn('price');
grid.clearSort(columnPrice);

Example: clear a specific column (getColumnById)

The getColumnById method is similar to getColumn.
Each column has an id. If not provided, it will be automatically generated based on the columnâ€™s index
or with a prefix like 'id-'.
IDs are helpful when two columns share the same data index.
In such cases, the grid uses IDs to distinguish them. 


const columnPrice = grid.getColumnById('price-2');
grid.clearSort(columnPrice);

Example: clear sorting for multi sorted grid


const columnPrice = grid.getColumn('price');
grid.clearSort(columnPrice, true);

multiSort

(column: object, dir: 'ASC' | 'DESC'): void

Sorting grid by several columns.

To sort by a specific column, you need to pass the column object.
Columns can be retrieved using getColumn or getColumnById.
Use the multi argument when applying multi-column sorting.
If you want to sort multiple columns at once, you can also use the multiSort method.

Example: Multi sorting


const columnBrand = grid.getColumn('brand');
grid.sort(columnBrand, 'DESC', true);
...
const columnModel = grid.getColumn('model');
grid.sort(columnModel, 'ASC', true);

sort

(column: object, dir: 'ASC' | 'DESC', multi?: boolean): void

Sorts the grid by a given column and direction.
Keep in mind multi-column sorting.
If the multi parameter is omitted, all other sorters will be cleared.

To sort by a specific column, you need to pass the column object.
Columns can be retrieved using getColumn or getColumnById.
Use the multi argument when applying multi-column sorting.
If you want to sort multiple columns at once, you can also use the multiSort method.

Example: ASC sort


const columnPrice = grid.getColumn('price');
grid.sort(columnPrice, 'ASC');

Example: DESC sort


const columnPrice = grid.getColumn('price');
grid.sort(columnPrice, 'DESC');

Example: Multi sorting


const columnBrand = grid.getColumn('brand');
grid.sort(columnBrand, 'DESC', true);
...
const columnModel = grid.getColumn('model');
grid.sort(columnModel, 'ASC', true);
