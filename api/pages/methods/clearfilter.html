clearFilter(column: object, operator: '<'|'>'|'<='|'>='|'='|'!='): void

Clear filter from grid

Filter operators

There are several operators that possible to use in filter field.
They are: <,>,<=,>=,=,!=

Example: Clear all filters


grid.clearFilter();

Example: Clear all filters for column


const columnPrice = grid.getColumn('price');
grid.clearFilter(columnPrice);

Example: Clear operator filter for column


const columnPrice = grid.getColumn('price');
grid.clearFilter(columnPrice, '>');
