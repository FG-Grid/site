To get a reference to the grid widget from any place, use the method Fancy.get.

If the renderTo property was set, then use Fancy.get('my-grid')


new Grid({
  id: 'my-grid',
  renderTo: 'myGridReport',
  ...
});

function updateGridTitle() {
  const grid = Fancy.get('my-grid');

  grid.setData([...]);
}
