agFn

String | Function

Aggregation for row grouping.

There 2 ways of usage - predefined values and custom function

'sum' - for summary group values
'avg' - average value
'min' - min value
'max' - max value

Example


columns: [{
  index: 'price',
  filter: true,
  agFn: 'sum',
  type: 'currency'

Example


columns: [{
  index: 'price',
  filter: true,
  agFn(values) => values.reduce((sum, value) => sum + value, 0),
  type: 'currency'
