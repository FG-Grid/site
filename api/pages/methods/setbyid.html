setById(id: string, index: string|object, value:string|number|boolean|undefined|null): void

Change data item by id.

Flashing changes

By default, changes will be flashed.
Grid parameter flashChanges is true by default.
For flashing, parameter flashChangesColors is used.
First color for usual cell, second color to selected rows.

flashChanges: true,
flashChangesColors: ['#e0e5e9','beige']

Example: change one option


grid.setById('id-0', 'price', 45000);

Example: change several options


grid.setById('id-2', {
  price: 75000,
  brand: 'Toyota',
  model: 'Land Cruiser',
  year: 2025,
  country: 'Japan'
});
