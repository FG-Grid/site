defaultColumnWidth

Number

Default column width if no width for column was provided.

 120

defaultRowGroupSort

String

Possible values: asc-string, desc-string, asc-amount, desc-amount.

Default row group sorting.

 'desc-amount'

editorEnterAction

String

This option defines behavior on key press ENTER in cell editor.
By default, it will close editor and focus back edited cell.
With values right or bottom it will focus next cell and
open editor for it.

Possible values: stay, right, bottom.

 'stay'

Function

One of the way to style.

It returns row css classnames based on params

 

new Grid({
   rowCls(params){
     if(params.item.price > 75000){
       return ['high-price'];
     }
   },
...

rowStyle

Function

One of the way to style.

It returns row styles based on params

 

new Grid({
   rowStyle(params){
     if(params.item.price >= 30000 && params.item.price <= 35000){
       return {
         'background-color': 'rgba(111, 178, 112, 0.2)'
       }
     }
   },

rowClsRules

Function

One of the way to style.

It uses interesting approach of applying css classes if condition is true

 

new Grid({
   rowClsRules: {
    'low-price': params => params.item.price < 30000
  }
