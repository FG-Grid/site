rowGroupType

String

Possible values: row, column

For row group type, full row can have only group value

For column group type, each column can have values and also use aggregations over column property agFn

Order column type is limited for row grouping.
Since there are 2 ways to display row grouping, rowGroupType: 'row' and 'column'.
Order column type is supported for option rowGroupType equals to 'column'.

Default


row
