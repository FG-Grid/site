showColumn(column: Object, animate?: boolean)

Show column
If column was hidden before over property hidden or over method hideColumn or manually,
it is possible to show it manually or over method showColumn.
Hiding columns could be done with animation.
By default animate is false.

Example


const columns = [...];

const grid = new Grid({
  ...
  columns,
  ...

grid.showColumn(columns[1], true);
